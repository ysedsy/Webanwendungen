## Frontend lokal mit Express starten

Damit Includes wie Header und Footer ohne file:// CORS-Fehler funktionieren, das Frontend ueber HTTP starten:

1. Abhaengigkeiten installieren
	npm install
2. Server starten
	npm start
3. Im Browser oeffnen
	http://localhost:3000
